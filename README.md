# 📚 Início dos Estudos em Java

Este repositório marca o início da minha jornada no estudo da linguagem **Java**, com foco em desenvolvimento **Back-End**.

## 🚀 Conteúdos estudados até o momento

Durante as primeiras aulas, estudei os fundamentos da **Programação Orientada a Objetos (POO)**, incluindo:

* Classes
* Objetos
* Atributos
* Métodos
* Uso da palavra-chave `this`

Esses conceitos são a base para entender como estruturar sistemas utilizando o paradigma orientado a objetos.

## 🎯 Objetivo

Construir uma base sólida em **Java e POO**, evoluindo gradualmente para o desenvolvimento de aplicações **Back-End**, APIs e integração com banco de dados.

## 🛠 Tecnologias utilizadas

* Java
